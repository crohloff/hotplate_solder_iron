# hotplate_solder_iron
SMD solder station based on old hot iron
inspired by [makermoekoe](https://github.com/makermoekoe/Hotplate-Soldering-Iron)

## HW BOM
- old hot iron
- 230V/5V power supply
- SSR-25 DA
- 10k potentiometer (through hole)
- 10k resistor (0805 SMD)
- button / PTS645 or similar (SMD)
- 0,96" OLED (SSD1306/1305)
- MAX6675 HW-550 (k-type)

## FW req
- arduino software
- U8g2 lib by oliver
- MAX6675 lib by Adafruit
